## Purpose

The bundled example topic that ships with the template so the reports site has a
real, rendered report to show out of the box. Its report is synthesized from the
sample-session findings under `reports/_meta/sample-session/` (the findings JSON is
not committed here, so the deterministic counts below read zero). In an
instantiated harness, run `/start` to research your own first topic, which replaces
this example.
