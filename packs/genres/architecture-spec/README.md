# architecture-spec

An AI-ready architecture-specification genre (arc42/C4): cross-cutting entity types,
relationships, namespaces, decisions-with-alternatives, and EARS acceptance criteria a downstream
AI coding agent can execute.

This `genres` pack is documented in the MIF research-harness reference:

- **[architecture-spec — pack reference](https://modeled-information-format.github.io/research-harness-template/reference/packs/genres/#architecture-spec)** — its purpose, constraints, goals,
  and how to enable it.

**Dependencies:** the `ai-spec` channel (consumes this genre), `schemas/artifact.schema.json`

The pack source lives in this directory. It ships disabled; enable it with
`scripts/pack-toggle.sh architecture-spec on`. See the
[MIF research-harness docs](https://modeled-information-format.github.io/research-harness-template/) for the full pack catalog.
