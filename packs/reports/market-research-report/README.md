# market-research-report

This `reports` pack is documented in the MIF research-harness reference:

- **[market-research-report — pack reference](https://modeled-information-format.github.io/research-harness/reference/packs/reports/#market-research-report)** — its purpose, constraints, goals,

**Dependencies:** None beyond the core harness engine (`jq`, `python3`, `ajv`).

The pack source lives in this directory. It ships disabled; enable it with
`scripts/pack-toggle.sh market-research-report on`. See the
[MIF research-harness docs](https://modeled-information-format.github.io/research-harness/) for the full pack catalog.
