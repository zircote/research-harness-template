# pdf

This `channels` pack is documented in the MIF research-harness reference:

- **[pdf — pack reference](https://modeled-information-format.github.io/research-harness/reference/packs/channels/#pdf)** — its purpose, constraints, goals,
  and how to enable it.

**Dependencies:** pandoc + PDF engine, @mermaid-js/mermaid-cli (optional), jq

The pack source lives in this directory. It ships disabled; enable it with
`scripts/pack-toggle.sh pdf on`. See the
[MIF research-harness docs](https://modeled-information-format.github.io/research-harness/) for the full pack catalog.
