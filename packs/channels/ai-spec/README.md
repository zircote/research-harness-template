# ai-spec (channel pack)

Renders a topic's surviving findings into an AI-ready, agent-executable architecture spec — a
genre-shaping of the artifact -> Markdown pipeline (finding_refs -> evidence, goal checks -> EARS,
sections -> structure). Pairs with a spec-genre pack (architecture-spec / kiro-spec / feature-spec).
Optional, toggle-ready. See `skills/ai-spec/SKILL.md`.
